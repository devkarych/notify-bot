# Notify Bot (For RU users)

## App description

Simple notification service in Telegram.

## Stack

- Python 3.10+
- Aiogram 2.21+
- APScheduler
- PostgreSQL (SQLalchemy + asyncpg + alembic)
- PyLint as code analyzer
